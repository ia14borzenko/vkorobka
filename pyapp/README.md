# Vkorobka Python Test Application

Python приложение для тестирования компонентов системы vkorobka.

## Установка

```bash
pip install -r requirements.txt
```

## Использование

```bash
python main.py
```

### Динамик ESP32 (MAX98357A)

После запуска **win-x64** и подключения ESP32 по Wi‑Fi/TCP:

```bash
python speaker_cli.py --host 127.0.0.1 --port 1236 path/to/track.wav
```

Файл по возможности **48 kHz** моно/стерео; стерео усредняется в моно. Иначе будет ресэмплинг через **scipy**. В сеть уходит поток **MSG_TYPE_STREAM**, `stream_id=3`, несжатый PCM **24-bit LE** (как обратный канал к микрофону). Перед потоком отправляется **`dyn.on`**, после — **`dyn.off`**.

## Описание

Приложение последовательно тестирует два основных компонента системы:
1. win-x64 - локальный компонент
2. esp32 - через win-x64

Для каждого компонента отправляется тестовая JPG картинка (4 КБ), компонент должен отзеркалить её и вернуть обратно.

## Требования

- Python 3.8+
- Pillow для работы с изображениями
- UDP порт 1236 должен быть доступен для связи с win-x64 приложением
