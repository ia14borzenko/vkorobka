# Vkorobka Python Test Application

Python приложение для тестирования компонентов системы vkorobka.

## Установка

```bash
pip install -r requirements.txt
```

## Использование

```bash
python main.py
```

## Описание

Приложение последовательно тестирует два основных компонента системы:
1. win-x64 - локальный компонент
2. esp32 - через win-x64

Для каждого компонента отправляется тестовая JPG картинка (4 КБ), компонент должен отзеркалить её и вернуть обратно.

## Требования

- Python 3.8+
- Pillow для работы с изображениями
- UDP порт 1236 должен быть доступен для связи с win-x64 приложением
